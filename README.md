# 📊 Student Info Collector (Pandas)

A simple Python project that collects user information and displays it in a structured table using **Pandas DataFrame**.

This project is designed as a beginner-friendly exercise to practice:
- Python loops
- User input handling
- Data storage using lists
- Basic data analysis with Pandas

---

## 🚀 Features

- Take multiple user inputs (name & code)
- Store data in structured format
- Display data as a clean table
- Simple interactive console interface

---

## 🧠 How It Works

1. The program asks how many users you want to enter
2. It collects each user's:
   - Name
   - Code
3. Stores all data in a Pandas DataFrame
4. Displays the table if the user chooses to view it
